# blog
leadrbox blog


## running locally

you'll need the `octopress` gem installed so:

```bash
gem install octopress
```

and then

```bash
jekyll s
```


## Deploys

```bash
jekyll build
octopress deploy
````


## other stuff

This is running with the octopress 3.0. see [octopress](https://github.com/octopress/octopress) for more details
